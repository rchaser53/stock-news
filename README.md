# 株価ニュース取得プログラム

このプログラムは、設定ファイルに記載された会社の株価に関連する最新ニュースをChatGPT APIを使用して取得し、標準出力に表示します。

## 必要な環境

- Go 1.21以上
- OpenAI APIキー

## セットアップ

1. 依存関係をインストール:
```bash
go mod download
```

2. OpenAI APIキーを環境変数に設定:
```bash
export OPENAI_API_KEY="your-api-key-here"
```

## 設定ファイル

`config.yaml` に監視したい会社のリストを記載します:

```yaml
companies:
  - name: "トヨタ自動車"
    ticker: "7203"
  - name: "ソニーグループ"
    ticker: "6758"
```

- `name`: 会社名
- `ticker`: 証券コード

## 実行方法

```bash
go run main.go
```

または、ビルドして実行:

```bash
go build -o stock-news
./stock-news
```

3. 株価は前日比+2.3%で推移...

[2] ソニーグループ (6758)
------------------------------------------------------------
1. ゲーム事業の好調により四半期利益が予想を上回る...
2. 半導体部門への投資拡大を発表...
3. 株価は年初来で+15%の上昇...

## 注意事項

- OpenAI APIの利用には料金が発生します
- APIのレート制限にご注意ください
- ニュース情報はChatGPTの学習データに基づくため、リアルタイム性には限界があります
